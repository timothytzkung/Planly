# Planly
SFU Planly is a degree planning web app for SFU students. Students can upload their goSFU advising transcript pdf in order to show what they still need in a more digestible way. SFU Planly extracts completed courses, in progress courses, and program requirements in order to help students plan future semesters using live course outline data.

## Instructions:
1. Navigate to the root folder (not inside backend or frontend)
2. run `npm install` to download root dependencies
2. Run `npm run setup` to download project dependencies
3. Run `npm run start` to start development

## General Documentation
`npm run start` will run frontend and backend in parallel--don't need to change directories.
`npm run frontend` will run only the frontend
`npm run backend` will run only the backend
